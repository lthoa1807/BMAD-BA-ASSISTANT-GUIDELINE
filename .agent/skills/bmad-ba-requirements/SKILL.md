# BA Requirements

## Goal

Convert discovery and fit-gap findings into requirements that implementation teams can estimate, configure, build, test, and trace.

## Execution

Read `./workflow.md`, then execute only the current step. Load the next step only when the current step is complete.

## Output

Create traceable step records under `{ba_artifacts}/requirements/`, update `{ba_artifacts}/ba-worklog-index.md`, and save the requirements pack under `{ba_artifacts}/requirements/` with a descriptive filename such as `ba-requirements-pack.md`.
