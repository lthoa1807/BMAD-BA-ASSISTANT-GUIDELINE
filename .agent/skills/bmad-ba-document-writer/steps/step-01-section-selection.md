# Step 1: Section Selection

## Goal

Select the section or sections to write and confirm usable source inputs.

## Instructions

Review the document plan, template, and available source material. Confirm:

- Document name and target folder
- Section filenames to write or update
- Intended audience and tone
- Source inputs for each selected section
- Whether knowledge notes should be reused, cited, traced, or checked for conflicts
- Known assumptions, open questions, and review needs

If the user asks to reuse knowledge notes, require confirmed note paths or summaries and record the requested linkage type.

## Output Fragment

Create a section writing plan with selected sections, source inputs, assumptions, questions, and readiness for drafting.

## Step Documentation

Before moving to the next step, create or update `{ba_artifacts}/documents/<document-name>/writing/step-01-section-selection-notes.md` with:

- Workflow and step name
- Inputs reviewed
- Questions asked
- User responses and evidence
- Selected sections with `DOC-###` IDs where useful
- Knowledge reuse request if any
- Decisions with `DEC-###` IDs
- Assumptions
- Risks with `RISK-###` IDs
- Open questions with `Q-###` IDs
- Output fragment
- Next action

Update `{ba_artifacts}/ba-worklog-index.md` with the completed step, artifact path, key decisions, risks, open questions, and next action.
