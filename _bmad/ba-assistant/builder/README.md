# BA Assistant Builder

Working records for building and evolving the `ba-assistant` custom module.

Use this area to keep planning, decisions, notes, logs, and reviews separate from the runtime workflow files.

## Folders

- `plan/` - Version plans, implementation plans, and phased build plans.
- `decision/` - Design decisions and accepted/rejected options.
- `notes/` - Working notes, rough analysis, and discussion summaries.
- `log/` - Build progress records and implementation logs.
- `review/` - Review findings, validation notes, and readiness checks.

## Rule

Builder files document how the module is created. Runtime skill behavior should live in the workflow, step, agent, help, and module metadata files outside this folder.
