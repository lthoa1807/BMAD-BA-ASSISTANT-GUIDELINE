# BA Discovery

## Goal

Create a business discovery brief that gives the delivery team a shared, evidence-based view of scope and implementation context.

## Execution

Read `./workflow.md`, then execute only the current step. Load the next step only when the current step is complete.

## Output

Create traceable step records under `{ba_artifacts}/discovery/`, update `{ba_artifacts}/ba-worklog-index.md`, and save the discovery brief under `{ba_artifacts}/discovery/` with a descriptive filename such as `business-discovery-brief.md`.
