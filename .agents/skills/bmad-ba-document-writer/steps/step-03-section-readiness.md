# Step 3: Section Readiness

## Goal

Check drafted sections for readiness and update supporting registers.

## Instructions

Review the drafted sections and identify:

- Sections ready for stakeholder review
- Sections needing source evidence
- Sections blocked by decisions or SME answers
- Assumptions to add or update in `_assumptions.md`
- Open questions to add or update in `_open-questions.md`
- Evidence references to add or update in `_evidence-register.md` when useful

Do not perform a full document review in this workflow. Recommend `bmad-ba-document-review` when the user needs review findings.

## Step Documentation

Create or update `{ba_artifacts}/documents/<document-name>/writing/step-03-section-readiness-notes.md` with:

- Workflow and step name
- Inputs reviewed, including drafted sections
- Questions asked
- User responses and evidence
- Readiness findings with `DOC-###` IDs
- Updated supporting register paths
- Decisions with `DEC-###` IDs
- Assumptions
- Risks with `RISK-###` IDs
- Open questions with `Q-###` IDs
- Recommended next workflow

Update `{ba_artifacts}/ba-worklog-index.md` with workflow completion status, artifact paths, carried-forward risks, open questions, and recommended next workflow.

## Completion

Report the drafted section paths and recommend `bmad-ba-document-review` when the sections need quality and evidence review.
