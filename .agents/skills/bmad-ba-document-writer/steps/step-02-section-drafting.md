# Step 2: Section Drafting

## Goal

Draft the selected Markdown section files using available evidence and explicit assumptions.

## Instructions

For each selected section:

- Follow the template guidance when available
- Use business language appropriate for the audience
- Keep source-backed statements distinct from assumptions
- Add placeholders for missing facts instead of inventing content
- Add citations or knowledge references only when explicitly requested
- Capture open questions inline only when they help reviewers resolve the draft

Use numbered filenames such as `01-overview.md`, `02-scope.md`, or the filenames from the template.

## Output Fragment

Create or update the selected section files and summarize what was drafted, assumed, or left unresolved.

## Step Documentation

Before moving to the next step, create or update `{ba_artifacts}/documents/<document-name>/writing/step-02-section-drafting-notes.md` with:

- Workflow and step name

- Questions asked

- Drafted section paths with `DOC-###` IDs
- Decisions with `DEC-###` IDs
- Assumptions
- Risks with `RISK-###` IDs
- Open questions with `Q-###` IDs
- Output fragment
- Next action

Update `{ba_artifacts}/ba-worklog-index.md` with the completed step, artifact path, key decisions, risks, open questions, and next action.
