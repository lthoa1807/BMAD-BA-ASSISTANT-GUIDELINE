# Short Document Template

Packaged generic short document template resources for `bmad-ba-short-doc-production`.

Copy these files to `{ba_artifacts}/documents/template-doc/short-doc/` before producing a runtime short document. Runtime copies may be adapted for the project, product, customer, document purpose, or preferred document language.

Required files:

- `00-document-template.md`
- `writing-rules.md`
- `review-checklist.md`
- `generation-workflow.md`
- `section-map.md`
- `section-file-list.md`

Do not add real customer documents, binary sample files, or generated final outputs to this packaged template.
