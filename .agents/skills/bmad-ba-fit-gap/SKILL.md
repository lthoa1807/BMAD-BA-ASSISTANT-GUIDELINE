# BA Fit Gap

## Goal

Create a fit-gap matrix that protects standard fit while making genuine gaps, decisions, and implementation risks explicit.

## Execution

Read `./workflow.md`, then execute only the current step. Load the next step only when the current step is complete.

## Output

Create traceable step records under `{ba_artifacts}/fit-gap/`, update `{ba_artifacts}/ba-worklog-index.md`, and save the fit-gap matrix under `{ba_artifacts}/fit-gap/` with a descriptive filename such as `ba-fit-gap-matrix.md`.
