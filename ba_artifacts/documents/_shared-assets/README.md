# Shared Assets

Lưu tài nguyên dùng chung cho các tài liệu.

## Nên lưu tại đây

- Screenshot.
- Logo hoặc icon.
- Diagram.
- File phụ lục.
- Hình ảnh dùng lại ở nhiều tài liệu.

## Gợi ý tổ chức

```text
_shared-assets/
  <document-name>/
    images/
    diagrams/
    attachments/
```

Ví dụ:

```text
_shared-assets/
  fbo-user-guide/
    images/
      login-screen.png
```
