# Sample Documents

Lưu các tài liệu mẫu dùng để phân tích format, cấu trúc, giọng văn và quy tắc trình bày.

## Nên lưu tại đây

- Tài liệu hướng dẫn sử dụng bạn đã viết trước đây.
- File mẫu ở định dạng `.docx`, `.pdf`, `.md`, `.xlsx`, `.pptx` nếu có liên quan.
- Ảnh chụp màn hình của tài liệu mẫu nếu chưa có file gốc.

## Không nên lưu tại đây

- Bản nháp tài liệu mới.
- Template đã được chuẩn hóa.
- Bản final dùng để phát hành.

## Gợi ý tổ chức

```text
sample-doc/
  <system-or-domain>/
    <document-name>.<ext>
```

Ví dụ:

```text
sample-doc/
  FBO/
    fbo-user-guide-sample.docx
```
